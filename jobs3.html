Job Overview

TECHYA is a small-scale IT education organization specializing in online courses and training programs for tech skills. We offers interactive learning modules, certification programs, and hands-on projects to help students gain practical experience in fields such as software development, cybersecurity, cloud computing, and data science.

As a Frontend Developer at Techya, you will be responsible for designing and optimizing the user-facing side of the platform while ensuring smooth integration with backend services and DevOps pipelines. As part of the DevOps Team, your role extends beyond traditional frontend work to include collaboration on deployment automation, performance monitoring, and UI optimizations. You will work closely with backend developers and DevOps engineers to create scalable, responsive, and high-performance web applications.

Responsibilities

  • ADevelop and maintain responsive web applications using React, Vue.js, or similar frameworks
  • Optimize frontend performance for seamless user experience and fast load times
  • Work with the DevOps team to implement CI/CD pipelines for frontend deployment
  • Ensure smooth API integration and communication with backend services
  • Debug and resolve frontend issues while ensuring cross-browser compatibility
  • Collaborate with designers and backend engineers to improve UI/UX and data flow
  • Contribute to automation and monitoring of frontend performance using DevOps tools/li>

Required Qualifications

  • Bachelor’s degree in Computer Science, Software Engineering, or related field (or equivalent experience)
  • 1+ years of experience as a Frontend Developer in a production environment, preferably education-related.
  • Proficiency in HTML, CSS, JavaScript, and modern frontend frameworks (React, Vue.js, Angular)
  • Experience with RESTful APIs and backend integration
  • Familiarity with DevOps tools like Docker, Kubernetes, and CI/CD pipelines (Jenkins, GitHub Actions)
  • Knowledge of performance monitoring tools (Lighthouse, Web Vitals)
  • Understanding of security best practices for frontend applications
  • Experience with testing frameworks like Jest or Cypress is a plus
