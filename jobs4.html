Job Overview

TECHYA is a small-scale IT education organization specializing in online courses and training programs for tech skills. We offers interactive learning modules, certification programs, and hands-on projects to help students gain practical experience in fields such as software development, cybersecurity, cloud computing, and data science.

As a Backend Developer at Techya, you will be responsible for building and maintaining the core server-side logic, APIs, and database infrastructure that power the online learning platform. As part of the DevOps Team, your role involves not only writing efficient and secure code but also collaborating on automation, monitoring, and deployment strategies. You will work closely with frontend developers and DevOps engineers to ensure smooth integration and high system availability.

Responsibilities

  • Develop and maintain backend services using Node.js, Python, or other backend technologies
  • Design, implement, and optimize databases (SQL & NoSQL) for scalability
  • Build RESTful APIs and GraphQL endpoints for frontend and mobile applications
  • Collaborate with DevOps engineers to automate deployments, monitoring, and scaling
  • Implement authentication, authorization, and security best practices
  • Optimize backend performance and troubleshoot bottlenecks
  • Ensure cloud-based deployments are efficient and cost-effective (AWS, Google Cloud, or Azure)
  • Write and maintain backend documentation and technical guides

Required Qualifications

  • Bachelor’s degree in Computer Science, Software Engineering, or equivalent experience
  • Strong proficiency in backend programming languages (Node.js, Python, Ruby, or Java)
  • Experience with cloud platforms (AWS, Google Cloud, Azure) and serverless architecture
  • Experience working with cloud-based databases (AWS RDS, Google Cloud SQL, Azure SQL, etc.) and database management software, such as Oracle, SQL Server, or MySQL.
  • Knowledge of containerization (Docker, Kubernetes) and infrastructure as code (Terraform)
  • Familiarity with CI/CD workflows and deployment automation tools
  • Experience with relational and non-relational databases (PostgreSQL, MongoDB, MySQL)
  • Strong understanding of API security (OAuth, JWT) and data encryption
  • Experience with logging, monitoring, and performance tools like Prometheus, Grafana, or ELK stack is a plus
